4010 Grade Calculator

What does this do?
  • Calculates your total extra credit, total homework, quiz, test, and project grades, and overall grade.
  • Does not save your scores in any way.
  • Calculations are only done on this browser.

How do I use this?

You only need to input the exact scores you have in Canvas (e.g., 90, 100, etc.). Decimals and percents will not work. Project grades have been changed, so the max score for a project is 15.
